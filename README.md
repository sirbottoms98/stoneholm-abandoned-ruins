# Stoneholm Abandoned Ruins

A datapack rebalance for the Stoneholm mod that transforms underground villages into rare abandoned ruins.

Designed primarily for large modpacks and multiplayer servers, this datapack reduces villager density, limits structure complexity, disables underwater villages, and favors abandoned corridors, crypts, and ruined points of interest to improve both performance and atmosphere.

# Features
1. Makes Stoneholm villages significantly rarer
2. Reduces structure size and expansion
3. Disables underwater village generation
4. Greatly reduces villager spawns
5. Reduces villager-related POIs and room density
6. Increases abandoned corridors, unfinished rooms, and crypt generation
7. Reduces carpet and decorative clutter
8. Improves server TPS and chunk generation stability

# This datapack is intended to help reduce:

1. Villager AI lag
2. Excessive chunk generation load
3. Worldgen instability
4. Waterlogging and flooded underground structures
5. TPS drops during exploration
6. Large underground settlement sprawl

# Compatibility
Minecraft 1.21.x
NeoForge
Stoneholm

## Installation

Place the datapack zip into:

world/datapacks/

Then restart the server or run:

/reload

# Will only work on new world generation!
